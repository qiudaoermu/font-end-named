# 前端开发常用命名

## 函数-func 

跳转路由: **goto()**



配置命名: ***xxx-setting***

初始化请求函数: **initEditor()**

拉取初始请求: **queryDetailById(id)**

## 字段-field

数据源: **dataSource** n

参数: **params** n

表单: 提交集合 **fileds** n

分离: **exract** v

队列: **queue** n

迭代器: **iterator** n

## 文件夹-folder
脚本: **scripts**

文档: **docs**

打包输出文件夹: **build**

工具: **util**

枚举: **enum**
